## 一、APP 图标 (App Icon)

### Android

- **路径**：`android/app/src/main/res/`
- **Adaptive Icon**（推荐）：
  - 前景图：`mipmap-xxxhdpi/ic_launcher_foreground.png`（建议 432×432 安全区，整体 108×108 dp）
  - 背景图：`mipmap-xxxhdpi/ic_launcher_background.png` 或单色 `color/ic_launcher_background`
- **传统图标**（各分辨率）：
  - `mipmap-mdpi/ic_launcher.png` (48×48)
  - `mipmap-hdpi/ic_launcher.png` (72×72)
  - `mipmap-xhdpi/ic_launcher.png` (96×96)
  - `mipmap-xxhdpi/ic_launcher.png` (144×144)
  - `mipmap-xxxhdpi/ic_launcher.png` (192×192)
- **圆角**：`ic_launcher_round.*` 同上尺寸，用于部分启动器。
- **清单**：`AndroidManifest.xml` 中 `android:icon`、`android:roundIcon` 已指向 `@mipmap/ic_launcher`，替换上述资源后无需改代码。

### iOS

- **路径**：`ios/项目名/Images.xcassets/AppIcon.appiconset/`
- **尺寸**：按 Contents.json 中配置，通常需 1024×1024（App Store）、20@2x、20@3x、29、40、60@2x、60@3x、76、83.5@2x 等。
- 使用 Xcode 打开 `Images.xcassets`，将设计师提供的各尺寸拖入对应槽位即可。

### 设计交付物

- 1024×1024 主稿（App Store / 设计源）
- Android 前景/背景分层（若用 Adaptive Icon）
- 通知栏小图标（单色剪影，可选）

## 二、启动页 (Splash Screen)

### 设计规范（执行方案）

- 时长约 2.5–3s，最长 4s。
- 阶段：沉寂(黑+小光点) → 燃起(光扩散+Logo) → 照亮(Slogan「看见微光，就是看见希望」) → 淡出进入首页。
- 背景深色 #0F172A，不受系统浅色模式影响。

### Android

- **Android 12+**：使用 `androidx.core:core-splashscreen`，在 `android/app/src/main/res/values/` 下配置 `windowSplashScreenBackground`（如 `#0F172A`）、`windowSplashScreenAnimatedIcon`（可为 drawable 或 Lottie）。
- **Lottie 动效**：将 Lottie JSON 放入 `android/app/src/main/assets/`，在 Splash 中通过 Lottie 库播放；或由原生实现「光点→扩散→Slogan」的动画。
- **旧版**：用 `res/drawable/launch_screen.xml` + `styles.xml` 中 `windowBackground` 指向该 drawable，实现静态 Splash；动效需在 JS 层用全屏组件 + 2.5s 后导航到主栈实现（有短暂白屏风险，不推荐为首选）。

### iOS

- **LaunchScreen.storyboard**：在 `ios/项目名/LaunchScreen.storyboard` 中设置背景色 #0F172A、中央 Logo 图片；复杂动效需在首屏 JS 中做「伪 Splash」全屏组件，或使用 `react-native-bootsplash` 等库配合 Lottie。
- **静态图**：可放置一张「微光扩散+Slogan」的合成图作为 Launch Image，避免白屏。

### React Native 层（可选）

- 若原生 Splash 仅做静态深色+Logo，可在 App 根组件中判断「是否首次挂载」，若首次则渲染全屏「Splash 动效组件」（Lottie + Slogan），2.5s 后隐藏并显示 MainNavigator，以实现与方案一致的「燃起→照亮→入场」体验。
- 推荐库：`lottie-react-native` 播放 Splash 动画 JSON；动画资源需设计师按执行方案输出。
