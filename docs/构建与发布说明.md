# 点亮微光 · 构建与发布说明

## 一、应用名称

- **显示名称**：点亮微光
- **Android**：`android/app/src/main/res/values/strings.xml` → `app_name`
- **app.json**：`displayName: "点亮微光"`

## 二、环境要求

- Node.js 18+
- Java 17（JDK，用于 Android 构建）
- Android SDK（含 platform-tools、build-tools）
- 设置 `JAVA_HOME` 指向 JDK 安装目录

## 三、Android 构建

### 生成 Release APK

```bash
cd android
./gradlew assembleRelease
```

Windows：

```powershell
cd android
.\gradlew.bat assembleRelease
```

输出路径：`android/app/build/outputs/apk/release/app-release.apk`

### 前置条件

1. 配置 `JAVA_HOME`：
   ```powershell
   # 示例（按实际路径修改）
   $env:JAVA_HOME = "C:\Program Files\Java\jdk-17"
   ```

2. 配置 Android SDK：
   ```powershell
   $env:ANDROID_HOME = "C:\Users\你的用户名\AppData\Local\Android\Sdk"
   ```

## 四、iOS 构建（macOS）

```bash
cd ios
xcodebuild -workspace 项目.xcworkspace -scheme 项目 -configuration Release
```

需 Xcode 与 CocoaPods 环境。

## 五、验证期打包建议

- 使用 `assembleRelease` 生成的 APK 可进行内部测试（TestFlight / 直装）
- **正式上架应用市场**需配置 Release 签名（keystore），详见 **`华为应用市场上架指南.md`**
