# 第一阶段·核心攻坚执行方案

> **目标**：通过视觉符号与动态交互，将「光之宇宙」哲学理念具象化，完成从「工具」到「精神家园」的感官转化。  
> **范围**：APP 图标、原生启动页（Splash）、「灯火愈明」数据可视化组件。

---

## 一、APP 图标设计 (App Icon Design)

**设计理念**：「微光破晓，双手呵护」——在应用商店中体现希望与陪伴。

### 方案 A：【微光初现】（推荐）

- **视觉**：深邃夜空蓝（#0F172A）背景，中心为极简古风灯笼/烛火，温暖橙黄光晕（#FFD700 → #FF6347 渐变）微微扩散、呼吸感。
- **隐喻**：最深黑夜中微光仍能照亮前方；家长对孩子的守护。
- **风格**：扁平化 + 轻微弥散光影，纯净感。

### 方案 B：【薪火相传】

- **视觉**：两只抽象柔和的手掌心相对，托举一簇火焰；火焰由底红过渡到顶金白，动态上升。
- **隐喻**：传灯人身份，传递与承接。
- **风格**：插画风，温暖治愈。

### 输出规格

| 平台 | 规格 |
|------|------|
| iOS | 1024×1024 (App Store)，@2x/@3x 各尺寸 |
| Android | Adaptive Icon（前景+背景分层），512×512 |
| 通用 | favicon、通知栏小图标（单色剪影） |

**动效预留**：支持动态图标的系统可做火光轻微呼吸闪烁。  
**验收**：在深色/浅色桌面背景下做对比测试，确保辨识度。

## 二、原生启动页 (Native Splash Screen)

**设计理念**：「从黑暗到光明的仪式感」——启动页是进入心流状态的心理开关。

### 动效脚本 (2.5s–3s，最长 4s)

| 阶段 | 时间 | 内容 |
|------|------|------|
| **沉寂** | 0s–0.5s | 全黑 #000000；中央极小暖黄光点（约 2px）；可选极轻风声/划火柴声 |
| **燃起** | 0.5s–1.5s | 光点扩大、火焰粒子；背景由黑渐变至 #0F172A；Logo 随火光浮现、轻微上浮 |
| **照亮** | 1.5s–2.5s | 光晕柔和扩散（Blur）；Slogan「看见微光，就是看见希望」自下淡入上移，纤细衬线/圆体，纯白带微光；可选风铃/钢琴单音 |
| **入场** | 2.5s–结束 | 画面淡出，过渡至首页【心灯导引】 |

### 技术要点

- **原生优先**：iOS LaunchScreen.storyboard + UIView 动画，或 Android SplashScreen API (12+) + Lottie，零卡顿、无白屏。
- **暗黑**：强制深色背景，不受系统主题影响。
- **性能**：动画资源压缩，避免拖慢启动。

## 三、「灯火愈明」数据可视化组件

**设计理念**：「拒绝冰冷图表，呈现生命生长」——数据即光的生长史。

### 核心视图

- **心灯模式**：中央虚拟灯笼；亮度 ∝ 连续学习/打卡；光晕 ∝ 亲子沟通质量；火焰色随情绪（红=焦虑/活跃，金=平和，蓝=冷静）。
- **星河模式**：横向时间轴，每日一星；完成任务日点亮，连续日形成发光连线；里程碑为超新星式闪光。

### 交互

- 触摸：流光拖尾；点击某日星光 → 弹窗「光影日记」。
- 背景：缓慢流动极光/粒子，象征时间与生命流动。

### 数据映射

| 数据维度 | 可视化表现 | 用户感知 |
|----------|------------|----------|
| 连续打卡天数 | 火焰高度 / 星星连长度 | 坚持让火更旺 |
| 课程完成度 | 灯油充盈度 / 星云密度 | 储备足够能量 |
| 亲子互动评分 | 光色温暖度（色温） | 关系更温暖 |
| 复学里程碑 | 高亮特效（如烟花） | 高光时刻 |

### 技术建议

- **绘图**：React Native 可用 Rive/Lottie 做复杂动效，Victory/自定义 SVG 做数据图形。
- **性能**：限制粒子数，低端机降级为静态渐变。
- **反馈**：数据更新时火焰/光晕用弹性动画，即时反馈。

### 色彩规范

- 背景：Deep Space Blue #0F172A；渐变层 #0F172A → #1E293B 径向。
- 数据光：起步 #FDE68A / 成长 #FB923C / 里程碑 #FACC15 + #FFFFFF / 稳定点缀 #67E8F9。
- 光晕：多层外发光，混合模式 Screen/Add，模糊半径约 50%–100% 元素大小。

### 动效逻辑

- 加载：从左至右依次点亮（墨水晕染/光线扫描）。
- 数值变化：弹性缓冲（Spring），模拟火焰跳动。
- 点击：涟漪或粒子炸裂 + 详情浮层。
- 呼吸：核心灯/星静止时轻微缩放(1.0↔1.05)与透明度，周期 3–4s。

### 适配

- 深色模式强制；浅色模式自动切深空蓝背景。
- 无障碍：用亮度与大小区分等级，不依赖单一颜色。

## 五、AI 绘图与设计规范（节选）

- **整体界面**：Mobile app UI, "Brightening Lights", dark #0F172A, 中心灯笼/星簇，体积光与粒子，火焰高度替代柱状图，星座连线替代折线，毛玻璃卡片。Ethereal, healing, hopeful。
- **心灯特写**：极简灯笼中心发光，火焰由底红至顶金白，外围 Bloom 光晕随数据变化，深色背景+浮尘。
- **星河时间轴**：横向发光星路，亮星=里程碑，暗星=普通日，连续日成星座线，触摸星爆裂小火花，暖金/橙/黄。
- **光色温度**：柔和发光球体场，色温由冷静蓝到温暖橙，毛玻璃面板+细白字体。

详细 Prompt、色彩/图形/动效参数见项目内「灯火愈明」设计规范与 Lottie 交付清单。
